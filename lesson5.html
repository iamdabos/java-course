Lesson 5: Operators

What Are Operators?

Operators are symbols that perform operations on variables and values.

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)

Comparison Operators

  • == Equal to
  • != Not equal to
  • < Less than
  • <= Less than or equal to
  • > Greater than
  • >= Greater than or equal to

Logical Operators

  • && Logical AND
  • || Logical OR
  • ! Logical NOT

Example Program

int a = 10;
int b = 5;
System.out.println(a + b);  // Addition
System.out.println(a > b);  // Comparison
System.out.println(a > 0 && b > 0);  // Logical
Task:

Create two numeric variables, perform arithmetic, comparison, and logical operations on them, and print the results.
