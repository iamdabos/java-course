Lesson 3: Your First Java Program

Java Program Structure

Every Java application starts execution from the main method. Java programs are written inside classes.

//A Java program to print Hello World!
      public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

Explanation

Running Your Program

After writing your code, compile it using javac HelloWorld.java and run it using java HelloWorld in your terminal.

Task:

Modify the program to print your name, then compile and run it to see the output.
