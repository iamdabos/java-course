Lesson 7: Methods & Arrays

Methods in Java

Methods are blocks of code that perform a specific task and can be reused throughout the program.

public int add(int a, int b) {
    return a + b;
}

int sum = add(5, 3);
System.out.println(sum);  // Output: 8

Arrays in Java

Arrays store multiple values of the same type in a single variable.

int[] numbers = {1, 2, 3, 4, 5};
for(int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

Enhanced For Loop

Java provides a simpler way to iterate over arrays:

for(int num : numbers) {
    System.out.println(num);
}
Task:

Create a method that multiplies two numbers. Store results in an array and print all the values using both standard and enhanced for loops.
