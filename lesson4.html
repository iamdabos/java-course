Lesson 4: Variables & Data Types

Variables in Java

Variables are used to store data in memory. Every variable has a name and a data type.

Primitive Data Types

  • int - stores integers (whole numbers)
  • double - stores decimal numbers
  • char - stores single characters
  • boolean - stores true/false values
  • byte, short, long, float - other numeric types with different sizes

Declaring and Initializing Variables

int age = 25;
double salary = 3500.50;
char grade = 'A';
boolean isStudent = true;

Type Casting

You can convert one type to another using type casting:

int x = (int) 3.14;  // converts double to int
