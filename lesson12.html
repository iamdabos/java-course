Lesson 12: Java Collections Framework

What is the Collections Framework?

The Java Collections Framework provides a set of classes and interfaces to store, manage, and manipulate groups of objects efficiently.

Core Interfaces

  • List: Ordered collection that allows duplicates
  • Set: Unordered collection that does not allow duplicates
  • Map: Stores key-value pairs

List Example (ArrayList)

import java.util.ArrayList;

ArrayList names = new ArrayList<>();
names.add("Alice");
names.add("Bob");

for(String name : names) {
    System.out.println(name);
}

Set Example (HashSet)

import java.util.HashSet;

HashSet numbers = new HashSet<>();
numbers.add(1);
numbers.add(2);
numbers.add(2); // Duplicate ignored

System.out.println(numbers);

Map Example (HashMap)

import java.util.HashMap;

HashMap scores = new HashMap<>();
scores.put("John", 90);
scores.put("Jane", 85);

System.out.println(scores.get("John"));

When to Use What?

  • Use List when order matters
  • Use Set when uniqueness matters
  • Use Map when mapping keys to values
Task:

Create a program that stores student names in a List, unique course names in a Set, and student scores in a Map. Display all values.
