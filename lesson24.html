Lesson 24: Hibernate & JPA

Hibernate is an ORM framework for mapping Java objects to database tables. JPA (Java Persistence API) standardizes this.

Key Concepts

  • Entity classes and annotations (@Entity, @Table)
  • Relationships: One-to-One, One-to-Many, Many-to-Many
  • HQL (Hibernate Query Language)

Task

Create a Hibernate entity for a "Student" table and save a student object to the database.
