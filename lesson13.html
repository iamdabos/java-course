Lesson 13: File Handling in Java

File handling allows Java programs to read from and write data to files.

Writing to a File


import java.io.FileWriter;

FileWriter writer = new FileWriter("data.txt");
writer.write("Hello Java");
writer.close();

Reading from a File


import java.io.File;
import java.util.Scanner;

File file = new File("data.txt");
Scanner sc = new Scanner(file);

while(sc.hasNextLine()){
    System.out.println(sc.nextLine());
}
sc.close();

Task

Create a file, write your name into it, then read and display the content.
