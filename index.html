Lesson 1: What Is Java?

Introduction

Java is a high-level, object-oriented programming language designed to be simple, secure, portable, and powerful. It is used to build web apps, mobile applications, and enterprise software systems. It allows developers to write programs that can run on different devices and operating systems without modification.

A Brief History of Java

Java was developed in 1991 by a team led by James Gosling at Sun Microsystems. It was originally intended for embedded systems but became widely adopted with the growth of the internet.

In 2010, Oracle Corporation acquired Sun Microsystems and continues to maintain and evolve Java with regular updates.

Why Java Was Created

  • To solve platform dependency issues
  • To reduce memory management complexity
  • To improve software security
  • To simplify application development

Key Features of Java

  • Platform Independent: Write Once, Run Anywhere (WORA)
  • Object-Oriented: Based on classes and objects
  • Simple: Cleaner syntax than C/C++
  • Secure: JVM sandbox and strong security model
  • Robust: Exception handling and type safety
  • Multithreaded: Supports concurrent execution

How Java Works

  1. Java source code is written in a .java file
  2. The compiler converts it to bytecode (.class)
  3. The JVM runs the bytecode on any supported system

Where Java Is Used

  • Android applications
  • Banking and financial systems
  • Enterprise applications
  • Web servers and APIs
  • Big data platforms

Java Editions

  • Java SE: Core Java programming
  • Java EE (Jakarta EE): Enterprise applications
  • Java ME: Embedded systems
