Lesson 10: Interfaces & Abstract Classes

Abstract Classes

An abstract class cannot be instantiated. It may contain abstract methods (without implementation) and concrete methods (with implementation).

abstract class Animal {
    abstract void sound();

    void sleep() {
        System.out.println("Sleeping...");
    }
}

Extending an Abstract Class

class Cat extends Animal {
    void sound() {
        System.out.println("Cat meows");
    }
}

Interfaces

An interface is a fully abstract structure that defines methods a class must implement. Java supports multiple inheritance through interfaces.

interface Flyable {
    void fly();
}

Implementing an Interface

class Bird implements Flyable {
    public void fly() {
        System.out.println("Bird is flying");
    }
}

Abstract Class vs Interface

  • Abstract class can have method implementations
  • Interface methods are abstract by default
  • A class can implement multiple interfaces
  • A class can extend only one abstract class
Task:

Create an abstract class Employee with an abstract method calculateSalary(). Create an interface Payable and implement both in a concrete class.
