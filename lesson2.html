Lesson 2: Java Development Environment

Understanding JDK, JRE, and JVM

To work with Java, it is important to understand its core components:

  • JDK (Java Development Kit): Tools needed to write and compile Java programs.
  • JRE (Java Runtime Environment): Environment required to run Java programs.
  • JVM (Java Virtual Machine): Executes Java bytecode on the host machine.

Installing Java

Java can be installed by downloading the JDK from Oracle or OpenJDK. After installation, you can verify the setup using the java -version command in the terminal or command prompt.

Setting up an IDE

Integrated Development Environments (IDEs) make coding easier. Popular Java IDEs include:

  • IntelliJ IDEA
  • Eclipse
  • VS Code with Java extensions

They provide features like code completion, debugging, and project management.

Task:

Install the JDK on your system, set up an IDE, and run the java -version command to confirm the installation.
