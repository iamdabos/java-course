Lesson 2: Java Development Environment

Introduction

Before writing and running Java programs, developers must first set up a proper Java development environment. The Java ecosystem includes several important components that allow programmers to write, compile, and execute Java applications.

Understanding how these components interact is essential for every Java developer. In this lesson we explore the three core parts of the Java environment: JDK, JRE, and JVM. We will also learn how to install Java and configure a development environment using popular Integrated Development Environments (IDEs).

Understanding JDK, JRE, and JVM

Java programs rely on three main components that form the foundation of the Java platform. Each component plays a specific role in the development and execution of Java applications.

JDK (Java Development Kit)

The Java Development Kit is the complete toolkit used by developers to build Java applications. It contains tools needed to write, compile, debug, and package Java programs.

  • javac – the Java compiler
  • java – command used to run programs
  • javadoc – generates documentation
  • jar – packages Java files

The JDK also includes the Java Runtime Environment (JRE), meaning it provides everything necessary to both develop and run Java programs.

JRE (Java Runtime Environment)

The Java Runtime Environment provides the environment required to run Java applications. It contains the libraries and files needed by the Java Virtual Machine to execute programs.

Unlike the JDK, the JRE does not include development tools such as the compiler. It is designed for users who only need to run Java applications.

JVM (Java Virtual Machine)

The Java Virtual Machine is responsible for executing Java bytecode. When a Java program is compiled, it is converted into bytecode rather than machine code.

The JVM reads this bytecode and converts it into instructions that the computer can execute. This process allows Java programs to run on different operating systems without modification.

Installing Java

Before writing Java programs, the Java Development Kit must be installed. Java can be downloaded from Oracle or through OpenJDK distributions.

  1. Visit the official Java download website
  2. Download the JDK for your operating system
  3. Run the installer and follow the instructions
  4. Complete installation

After installation you can verify the setup using the command:

java -version

If Java is installed correctly, the terminal will display the installed version of Java.

Setting up an IDE

Integrated Development Environments (IDEs) help developers write and manage code more efficiently. They provide tools such as code completion, debugging features, and project management.

Popular Java IDEs

  • IntelliJ IDEA
  • Eclipse
  • Visual Studio Code with Java extensions

Choosing a good IDE can significantly improve productivity when developing Java applications.

Install the Java JDK on your computer and verify the installation using the java -version command.
